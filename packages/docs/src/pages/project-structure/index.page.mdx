## User & Admin

โครงสร้างการเขียนทั้ง ***user*** และ ***admin*** จะถูกวางไว้เหมือนกัน แต่จะมีในส่วนของ `i18n` ที่ทำให้ในส่วนของ routes ต่างกัน
เพราะในส่วนของ ***user*** จะมี `[lang]` แต่ในฝั่งของ ***admin*** จะไม่มี ซึ่งเหตุผลตรงนี้คือการที่ฝั่ง ***user*** ต้องทำ SEO แต่ในฝั่ง ***admin*** นั้นไม่จำเป็นต้องทำ
